# better-evidence-project

## Setup

First install golang. The easiest way is with [Brew](brew.sh) using `brew install go`. If using Linux of Windows, follow the installation guide on the [Golang website](go.dev/dl).

For MacOS, next install air and tailwindcss using `make install-arm64` or `make install-x64` based on your Mac Architecture (x86_64 or Arm).
For Linux or Windows get the [tailwindcss binary](https://github.com/tailwindlabs/tailwindcss/releases/) and place it in the project's root directory, and install air following the [docs](https://github.com/air-verse/air).

## Development
Simply run `air` in your terminal. Changes will hot reload the page.
